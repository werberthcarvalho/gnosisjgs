Função Descrição Exemplo
print() Exibe dados na saída padrão (console). print("Olá, mundo!")
input() Lê texto digitado pelo usuário. nome = input("Seu nome: ")
len() Retorna o tamanho de sequência (str, list, tuple, dict, ...). len("Python") # 6
type() Retorna o tipo do objeto. type(3.14) # <class 'float'>
int() Converte para inteiro (quando possível). int("5") # 5
float() Converte para ponto flutuante. float("3.14") # 3.14
str() Converte valor para string. str(123) # "123"
list() Cria lista a partir de iterável. list("abc") # ['a','b','c']
dict() Cria dicionário (vazio ou a partir de pares/kwargs). dict(nome="Ana", idade=20)
max() Retorna o maior item de um iterável. max([1,5,3]) # 5
min() Retorna o menor item. min([1,5,3]) # 1
sum() Soma todos os elementos numéricos de um iterável. sum([1,2,3]) # 6
sorted() Retorna uma nova lista ordenada a partir de um iterável. sorted([3,1,2]) # [1,2,3]
range() Cria sequência numérica - útil em loops. list(range(5)) # [0,1,2,3,4]
enumerate() Itera retornando tuplas (índice, valor). for i,v in enumerate(["a","b"]): print(i,v)
zip() Agrupa iteráveis em pares (ou tuplas). list(zip([1,2],[3,4])) # [(1,3),(2,4)]
map() Aplica função a todos os elementos de um iterável. list(map(str,[1,2,3])) # ['1','2','3']
filter() Filtra elementos que satisfazem uma condição. list(filter(lambda x: x>2, [1,2,3,4]))
any() Retorna True se algum elemento for verdadeiro. any([0,1,0]) # True
all() Retorna True se todos os elementos forem verdadeiros. all([1,2,3]) # True
round() Arredonda números (padrão 0 casas ou n casas). round(3.1415,2) # 3.14
abs() Retorna valor absoluto. abs(-5) # 5
open() Abre arquivo (leitura/escrita). open('arq.txt','r')
help() Mostra documentação built-in de objetos/funções. help(len)
isinstance() Verifica se objeto é instância de um tipo/classe. isinstance(5,int) # True
format() Formata strings (método de string ou builtin). '{:.2f}'.format(3.1415) # '3.14'
